# Now Camp Candidate Assets

Generated concept assets for the P1 Now Camp / Leader Training direction.

These files are review candidates only. They are not runtime-integrated sprites
yet and should not be loaded by app code until the implementation plan defines
asset keys, target dimensions, slicing rules, Reduced Motion behavior, and QA
coverage.

Files:

- `now-camp-training-props-sheet.png` — field camp prop candidates.
- `now-camp-resonance-effects-sheet.png` — resonance, care, and training effect candidates.
- `now-camp-hero-layout-mockups.png` — layout reference mockups showing how overlays could sit on existing-style field heroes.

Generation mode:

- Built-in Codex image generation.
- Originals remain under `$CODEX_HOME/generated_images/`.

Runtime constraints:

- Do not replace existing field popover backgrounds.
- Do not replace species sprites.
- Do not add shop, chest, gem, ad, purchase, premium-currency, hunger, sickness,
  punishment, or decay imagery.
